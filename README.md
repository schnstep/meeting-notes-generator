# Meeting Notes Generator

Automatically creates structured meeting notes in Markdown format.

## Usage
'''bash
./meeting_notes.sh
'''

Enter meeting title when prompted. Script creates a dated file in '~/Documents/Meetings/'.

## Features

- Automatic date stamping
- Structured
- Action item checklist
- Auto-opens in default editor

## Installation
'''bash
chmod +x meeting_notes.sh
cp meeting_notes.sh ~/.local/bin/meeting-notes

'''
